Frameta

      
Uma ferramenta para fotógrafos que querem compartilhar não só a imagem, mas o processo por trás dela.


      

        

          
Como funciona

          
Faça upload de qualquer JPEG, PNG ou TIFF. O Frameta lê automaticamente os dados EXIF gravados pela câmera e os renderiza como overlay diretamente sobre a imagem, exportável nos principais formatos de redes sociais.

        

        

          
Privacidade

          
Nenhum arquivo é enviado a servidores. Todo o processamento acontece localmente no seu navegador. Dados de localização GPS nunca são exibidos.
